# RAG-APP
Production RAG app — upload CSV/Excel and ask AI questions about your data using FastAPI + ChromaDB + Claude

A full-stack Retrieval-Augmented Generation (RAG) application that lets you upload CSV or Excel files and query them using natural language. Built with FastAPI, ChromaDB for vector embeddings, and Anthropic Claude for AI-generated answers. Includes a clean web UI, REST API with Swagger docs, and Docker support for easy deployment.
